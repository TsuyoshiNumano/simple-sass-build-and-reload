# simple-sass-build-and-reload
boilerplate for simple sass build &amp; browser auto reload when file change

### Install and run
```
git clone git@github.com:TsuyoshiNumano/simple-sass-build-and-reload.git
```

```
$ npm i
```

```
$ npm run dev
```
